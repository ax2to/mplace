# GLOSARIO BASICO

## frontend
Parte visual de la app. Es lo que ves y usas en el navegador.

## backend
Parte que corre en el servidor. Recibe pedidos del frontend y responde con datos.

## API
Puente de comunicacion entre frontend y backend.

## endpoint
Direccion exacta de una API.
Ejemplo: `http://localhost:3100/api/products`

## base de datos
Lugar donde se guarda la informacion de forma organizada.

## tabla
Conjunto de datos dentro de la base de datos, como una hoja de calculo.

## registro
Una fila dentro de una tabla.

## puerto
Numero que usa un programa para comunicarse.
Ejemplo: backend en puerto `3100`.

## localhost
Tu propia computadora vista como servidor local.
